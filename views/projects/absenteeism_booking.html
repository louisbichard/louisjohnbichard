Absenteeism Booking

Angular demo application

A demo piece created in a short time frame for part of a recruitment process. Objectives were largely centered around interface concepts.

Built to the specification of the organisation with their current technology stack.

Technologies

  • Angular
  • SASS
  • FullCalendar

First deliverable

Live demo

Github

Features

  • Booking via calendar
  • Booking across different types (colour coded)
  • Clashing algorithm created to highlight clashes and prompt user when entering a clash
  • Filtration based on users, types and time of booking
  • Mocked authentication for booking as different users
  • Mocked API requests logged for all bookings made and deleted
  • Minimal click interface
